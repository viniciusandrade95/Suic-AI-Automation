## Intenção

O dashboard não deve ser uma coleção de caixas genéricas. Ele deve funcionar como uma sala de comando viva:

```text
estado vivo + decisão principal + 7 frentes + chatbot + timeline + visual emocional
```

A pergunta central do dashboard é:

```text
O que devo fazer agora?
```
