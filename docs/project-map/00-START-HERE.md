# 🔥 Start Here — Mapa de 30 segundos

Este ficheiro é para abrir quando o projeto parecer grande demais.

## O que o TheOne é

O TheOne é o sistema operacional/CRM para pequenos negócios de serviços.

Ele concentra:

- tenants;
- utilizadores/staff;
- clientes;
- serviços;
- locais;
- marcações;
- booking público;
- onboarding;
- prebooking do assistente;
- dashboard operacional.

## O que olhar primeiro

```text
1. 🔥 AGORA
2. 🔴 BLOQUEADO
3. 🔵 PODE JÁ
4. ⚡ PARALELO
5. ✅ FEITO
6. ⚠️ RISCO
```

## Cockpit rápido

| Bloco | Pergunta | Deve mostrar |
|---|---|---|
| 🔥 AGORA | O que faço agora? | 1 ação principal, motivo e desbloqueio. |
| 🔴 BLOQUEADO | O que impede avanço? | Item travado + dependência em falta. |
| 🔵 PODE JÁ | O que está pronto? | Tarefas desbloqueadas e curtas. |
| ⚡ PARALELO | O que posso adiantar? | Trabalho que não bloqueia o núcleo. |
| ✅ FEITO | O que já existe? | Progresso visível para reduzir ansiedade. |
| ⚠️ RISCO | O que pode atrasar? | Gargalos, ambiguidade e dívida. |

## Leitura mental ideal

```text
Estou aqui.
Isto está feito.
Isto está bloqueado.
Esta é a próxima ação.
Estas coisas podem andar em paralelo.
Este é o risco principal.
```

🔥 **Se estiveres perdido:** abre `03-EPICS-AND-CHECKPOINTS.md` e procura o primeiro item com categoria `🧱 Foundation` ou `🔓 Unlocker` que esteja `🔵 pronto`.

## Anti-confusão

Não tentar ler tudo.

Usa esta ordem:

1. título;
2. símbolo;
3. estado;
4. dependência;
5. critério de conclusão.

## Legenda visual curta

| Símbolo | Significado | Uso no mapa |
|---|---|---|
| 🔥 | Fazer agora | Ação recomendada principal |
| ✅ | Feito | Já existe ou está validado |
| 🟡 | Em andamento / parcial | Existe, mas ainda precisa prova ou acabamento |
| 🔵 | Pronto para começar | Não está bloqueado |
| 🔴 | Bloqueado | Falta dependência anterior |
| ⚡ | Paralelo | Pode avançar sem travar o núcleo |
| 🧱 | Fundação | Sem isto o resto fica frágil |
| 🔓 | Desbloqueador | Ao concluir, abre outras frentes |
| 🎯 | Milestone | Marco de avanço visível |
| ⚠️ | Risco | Pode atrasar, quebrar ou confundir |
| ✨ | Melhoria | Ajuda, mas não bloqueia MVP |
| 👁️ | Verificação visual | Usar para revisar UI/UX |
| 🧪 | Validação | Teste, smoke, evidência ou checklist |
